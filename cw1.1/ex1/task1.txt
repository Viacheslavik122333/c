В системе массового обсуживания есть 3 пункта обслуживания, к которым стоит общая очередь клиентов (например, кассы в супермаркете). В начальный момент времени первые 3 клиента из очереди занимают пункты обслуживания, и затем по мере обслуживания очередные клиенты из очереди занимают освобождающиеся пункты.
Время обслуживание каждого клиента индивидуально, заранее известно и занимает целое число минут. Времена обслуживания клиентов (целые числа) записаны в виде последовательности в файле. Общее количество клиентов заранее не известно.
Требуется определить какой клиент обслуживался каким пунктом (кассой), т.е. надо вывести в файл копию исходной последовательности вместе с номером пункта (пункты нумеруются 0, 1, 2), т.е. по два числа в строке: время из исходной последова-тельности и номер соответствующего пункта. Последовательность разрешается читать только один раз. Сохранять ее целиком в массиве нельзя. При наличии некорректных состояний или ошибок ввода-вывода нужно выдать соответствующее сообщение.
